    # vue3_study_demo(自学vite2.0+vue3+ts)

    This template should help get you started developing with Vue 3 in Vite.

    # 初始化

    # 查询node version & npm version

    node -v 

    npm -v

    # 通过vite2.0 创建 vue3

    # 推荐方式
npm init vue@next

    # npm 7+, 需要额外的双横线 (目录结构和引入的包可能有所不同)
npm init @vitejs/app my-vue-app -- --template vue

    # Recommended IDE Setup

    VSCode (opens new window) + Volar (opens new window) (and disable Vetur).

    # Type Support for .vue Imports in TS

    Since TypeScript cannot handle type information for .vue imports, they are shimmed to be a generic Vue component type by default. In most cases this is fine if you don't really care about component prop types outside of templates.

    However, if you wish to get actual prop types in .vue imports (for example to get props validation when using manual h(...) calls), you can run Volar: Switch TS Plugin on/off from VSCode command palette.

    npm run build

    Less

    #less  vite2.0 （无需 webpack 和 各种loader）
npm i less --save-dev

    vite-plugin-svg-icons

    npm i vite-plugin-svg-icons -D

    配置 vite.config.ts

    import viteSvgIcons from 'vite-plugin-svg-icons';
import path from 'path'; // ts如果报错 npm i @types/node -D
export default () => {
  return {
    plugins: [
      viteSvgIcons({
        // 配置路劲在你的src里的svg存放文件
        iconDirs: [path.resolve(process.cwd(), 'src/assets/icons')],
        symbolId: 'icon-[dir]-[name]',
      }),
    ],
  };
};

    main.ts

    import 'virtual:svg-icons-register';

    tailwind.css(如果选择tailwind.css 使用less等css预处理器将显得没有必要)

    # 安装 tailwind 依赖
npm install -D tailwindcss@latest postcss@latest autoprefixer@latest
# tailwind 配置初始化
npx tailwindcss init -p
# 安装 暗黑模式 依赖
npm install tailwindcss-dark-mode --save-dev

    配置 tailwind.config.js

    darkMode: 'media',
variants: {
  extend: {      
    textOpacity: ['dark'],
    backgroundColor: ['dark'],
  },
},
plugins: [
  require('tailwindcss-dark-mode')()//加一个支持暗黑模式的插件tailwindcss-dark-mode
],

    配置main.ts

    import "tailwindcss/tailwind.css"
